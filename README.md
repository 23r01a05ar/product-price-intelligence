# Online Product Price Intelligence System

## Project Overview
The **Online Product Price Intelligence System** is a full-stack application that automatically identifies products from images and compares prices across multiple e-commerce platforms.
The system helps users find the **best available price** for a product by combining **computer vision** and **web scraping** techniques.

---

## Features
* Product recognition using image processing
* Automatic product search
* Price comparison across multiple platforms
* Web scraping for real-time price data
* React-based frontend interface
* Flask API backend

---

↓

↓

↓
Web Scraping APIs
↓

↓
Frontend Display

## Installation

### Clone the repository
```
git clone https://github.com/Springboard-Internship-2025/Online-Product-Price-Intelligence-System_Feb_Batch-8_2026.git
```

### Backend Setup
```
cd backend
pip install -r requirements.txt
python run.py
```

### Frontend Setup
```
cd frontend/frontend-app
npm install
npm start
```
